# Comparatives & Superlatives: more/most/-er/-est → กว่า/ที่สุด

## Overview
Comparisons in English use a dual system (-er/-est for short words, more/most for longer words). Thai is simpler and more consistent, using กว่า (gwàa) for comparatives and ที่สุด (thîi-sùt) for superlatives.

## Pattern Mapping

### Comparatives (comparing two things)

| English Pattern | Thai Pattern | Example |
|----------------|-------------|---------|
| Adj + **-er** + than | Adj + **กว่า** | taller than → สูง**กว่า** |
| **more** + Adj + than | Adj + **กว่า** | more beautiful than → สวย**กว่า** |
| **less** + Adj + than | Adj + **น้อยกว่า** | less interesting → น่าสนใจ**น้อยกว่า** |
| **as** + Adj + **as** | Adj + **เท่ากับ** / **พอๆ กับ** | as tall as → สูง**เท่ากับ** |

### Superlatives (the most)

| English Pattern | Thai Pattern | Example |
|----------------|-------------|---------|
| **the** + Adj + **-est** | Adj + **ที่สุด** | the tallest → สูง**ที่สุด** |
| **the most** + Adj | Adj + **ที่สุด** | the most beautiful → สวย**ที่สุด** |
| **the least** + Adj | Adj + **น้อยที่สุด** | the least interesting → น่าสนใจ**น้อยที่สุด** |

## Key Simplification for Thai Students
- English has TWO systems (-er/more, -est/most) — Thai has ONE (กว่า / ที่สุด)
- Thai ALWAYS puts the comparison word AFTER the adjective
- No need to memorize which adjectives use -er vs more

## Complex Comparison Structures

### "The more...the more..."
- English: "The harder you study, the better your grades will be."
- Thai: ยิ่งเรียนหนัก คะแนนก็ยิ่งดี

### "Not as...as..."
- English: "This book is not as interesting as that one."
- Thai: หนังสือเล่มนี้ไม่น่าสนใจเท่ากับเล่มนั้น

## Chunking Rules for Comparatives
- "more/less" + adjective = ONE `modifier` chunk
- "than" + comparison target = separate `prepositional` chunk
- "-er" adjective = `modifier` chunk
- "the most/-est" adjective = `modifier` chunk

## Common Thai Student Mistakes
1. **Double comparative**: "more better" instead of "better"
2. **Missing "than"**: "He is taller me" instead of "He is taller than me"
3. **Using "more" with short adjectives**: "more big" instead of "bigger"
4. **Missing "the" with superlatives**: "She is tallest" instead of "She is the tallest"
