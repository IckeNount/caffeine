# Question Formation: Subject-Verb Inversion vs. Thai Question Particles

## Academic Overview
English question formation is syntactically complex, requiring subject-auxiliary inversion, do-support, and wh-movement — three operations that have no parallel in Thai. In Thai, questions are formed by appending interrogative particles (ไหม, หรือ, หรือเปล่า) to a declarative sentence or by placing question words (อะไร, ที่ไหน, ใคร) in-situ (in the same position as the answer). English, by contrast, requires structural reorganization of the entire sentence.

This difference represents a significant syntactic hurdle for Thai learners (CEFR A2–B2). The Thai question system is additive (add a particle), while the English system is transformational (rearrange the sentence). Thai students frequently produce questions using declarative word order with rising intonation — a strategy that works in Thai but is ungrammatical in English.

The challenge is further compounded by the do-support rule: English requires an auxiliary verb ("do/does/did") in questions where none exists in the declarative form — a concept entirely foreign to Thai grammar. "You like coffee" → "**Do** you like coffee?" involves inserting a word that has no semantic content, purely for syntactic reasons.

CEFR suitability: Yes/No questions (A2), Wh-questions with be (A2–B1), Wh-questions with do-support (B1), Embedded/indirect questions (B2).

## English vs. Thai: Key Differences

### Thai Question System (No Inversion Required)
Thai forms questions through three simple strategies:

1. **Yes/No questions**: Add ไหม (mǎi) or หรือเปล่า (rǔue-bplào) at the end
   - คุณชอบกาแฟ**ไหม** (You like coffee + QUESTION PARTICLE)
   - Word order is identical to the statement

2. **Wh-questions**: Place the question word WHERE the answer would go (in-situ)
   - คุณไป**ไหน** (You go where?) — ไหน sits where the destination would be
   - คุณชื่อ**อะไร** (You name what?) — อะไร sits where the name would be

3. **Choice questions**: Use หรือ (rǔue = or)
   - คุณจะกินข้าว**หรือ**ก๋วยเตี๋ยว (You will eat rice or noodles?)

### English Question System (Inversion Required)
1. **Yes/No**: Auxiliary moves before subject → "**Do** you like coffee?"
2. **Wh-questions**: Wh-word moves to front + inversion → "**Where do** you go?"
3. **Be-questions**: "be" moves to front → "**Are** you happy?"

## Chunking Rules for LinguBreak

### Question Sentences in Chunks
When analyzing question-form sentences, apply these rules:

1. **Wh-words** (what, where, when, who, why, how): Label as `modifier` when they function as question adverbs, or as `subject`/`object` when they fill a noun role
   - "**What** did you buy?" → object: "What" (it's what was bought)
   - "**Where** do you live?" → modifier: "Where" (modifies location)
   - "**Who** broke the glass?" → subject: "Who" (the actor)

2. **Auxiliary inversion**: The auxiliary is part of the `verb` chunk
   - "What **did you buy**?" → verb: "did buy" (reconstruct as one verb unit) + subject: "you"
   - "**Are** you happy?" → verb: "Are" + subject: "you" + modifier: "happy"

3. **Do-support**: Treat "do/does/did" as part of the verb
   - "**Does** she **speak** Thai?" → verb: "Does speak" (combines into one verb concept)

### Critical Rule for Thai Reconstruction
In the Thai reordered version, remove the inversion. Reconstruct the sentence in SVO order with the question particle at the end:
- English: "Where does she work?" → Thai logic: "She works where?" → เธอทำงานที่ไหน

## Thai Logic Reconstruction

### Reconstruction Steps for Questions

1. **Yes/No questions**: Remove the fronted auxiliary, restore SVO order, add ไหม
   - "Do you like coffee?" → Reconstruct: "You + like + coffee + ไหม?"
   - Thai: คุณชอบกาแฟไหม

2. **Wh-questions**: Move the wh-word to where the answer would go (Thai in-situ)
   - "Where do you live?" → Reconstruct: "You + live + where?"
   - Thai: คุณอยู่ที่ไหน

3. **Who as subject**: No change needed — same order in both languages
   - "Who teaches this class?" → "Who + teaches + this class?"
   - Thai: ใครสอนชั้นเรียนนี้

4. **Complex wh-questions**: Restore full SVO + place wh-word at answer position
   - "What has she been studying since last year?" → "She + has been studying + what + since last year?"
   - Thai: เธอเรียนอะไรมาตั้งแต่ปีที่แล้ว

## Common L1 Interference Errors (Thai Students)

1. **Declarative word order for questions**: "You go where?" → "**Where do** you go?"
   - *Cause*: Thai in-situ question words. คุณไปไหน is perfectly grammatical Thai.

2. **Missing do-support**: "Where you live?" → "Where **do** you live?"
   - *Cause*: Thai has no do-support concept. The auxiliary "do" is semantically empty and feels unnecessary.

3. **No inversion with "be"**: "Where you are going?" → "Where **are you** going?"
   - *Cause*: Thai doesn't move any verb; it simply adds a question word at the answer position.

4. **Using rising intonation instead of structure**: "You like this?" → "**Do** you like this?"
   - *Cause*: In Thai, rising intonation alone can signal a question, especially in casual speech.

5. **Double marking questions**: "**Do** you **are** happy?" → "**Are** you happy?"
   - *Cause*: Confusion between do-support and be-inversion; Thai students learn both rules and apply both simultaneously.

## Golden Examples

1. **Simple Yes/No with do-support**:
   - "**Does** your mother **speak** English fluently?"
   - Core SVO (restored): "Your mother speaks English fluently"
   - Thai: แม่ของคุณพูดภาษาอังกฤษคล่องไหม

2. **Wh-question with complex verb**:
   - "**What have** you **been doing** since I last saw you?"
   - Core SVO (restored): "You have been doing what since I last saw you"
   - Thai: คุณทำอะไรมาบ้างตั้งแต่ที่ฉันเจอคุณครั้งล่าสุด

3. **Who as subject (no inversion needed)**:
   - "**Who** told you about the exam?"
   - Core SVO: "Who told you about the exam"
   - Thai: ใครบอกคุณเรื่องข้อสอบ

4. **Embedded/indirect question**:
   - "I don't know **where she lives**."
   - Note: The embedded question uses normal SVO order — NOT inversion
   - Thai: ฉันไม่รู้ว่าเธออยู่ที่ไหน
