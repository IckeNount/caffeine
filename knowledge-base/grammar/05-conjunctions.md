# Conjunctions & Compound Sentences: and/but/or/so/because

## Overview
Conjunctions connect clauses, phrases, or words. Thai has direct equivalents for most conjunctions, making this one of the easier grammar patterns. However, complex sentences with multiple conjunctions can still confuse Thai students.

## Coordinating Conjunctions (FANBOYS)

| English | Thai | Usage |
|---------|------|-------|
| **for** | เพราะว่า (phrór-wâa) | reason/cause |
| **and** | และ (láe) | addition |
| **nor** | และไม่ (láe mâi) | negative addition |
| **but** | แต่ (dtàe) | contrast |
| **or** | หรือ (rǔue) | alternative |
| **yet** | แต่ก็ (dtàe gôr) | unexpected contrast |
| **so** | ดังนั้น (dang-nán) | result |

## Subordinating Conjunctions

| English | Thai | Category |
|---------|------|----------|
| because | เพราะว่า (phrór-wâa) | cause |
| although / even though | แม้ว่า (máe-wâa) / ถึงแม้ว่า | concession |
| if | ถ้า (thâa) | condition |
| when | เมื่อ (mûea) / ตอนที่ (dtorn-thîi) | time |
| while | ขณะที่ (khà-nà-thîi) | simultaneous |
| before | ก่อนที่ (gòrn-thîi) | time sequence |
| after | หลังจากที่ (lǎng-jàak-thîi) | time sequence |
| until | จนกว่า (jon-gwàa) | time limit |
| since | ตั้งแต่ (dtâng-dtàe) | time start / cause |
| so that | เพื่อที่จะ (phûea-thîi-jà) | purpose |

## Key Difference: Clause Order

### English: Subordinate clause can come first or last
- "**Because** it was raining, I stayed home."
- "I stayed home **because** it was raining."

### Thai: Similar flexibility, but preference for cause-first
- **เพราะ**ฝนตก ฉันเลยอยู่บ้าน (Because rain fell, I stayed home) — more natural
- ฉันอยู่บ้าน**เพราะ**ฝนตก (I stayed home because rain fell) — also OK

## Chunking Rules for Conjunctions
- Conjunction + its clause = think about whether to split or combine
- Short conjunctions (and, but, or): Usually a connector between two chunks, not its own chunk
- Subordinating conjunctions + clause: The entire subordinate clause = ONE chunk, labeled `modifier`
- "because he was tired" → one `modifier` chunk
- Compound sentences joined by "and/but": Split into separate analyses of each clause

## Complex Conjunction Patterns

### "Not only...but also"
- English: "Not only did he pass, but he also got the highest score."
- Thai: เขาไม่เพียงแต่สอบผ่าน แต่ยังได้คะแนนสูงสุดอีกด้วย
- Chunk: "Not only...but also" connects two parallel ideas

### "Either...or" / "Neither...nor"
- English: "Either you study hard or you fail."
- Thai: ไม่เรียนหนัก ก็สอบตก (simpler structure in Thai)

## Common Thai Student Mistakes
1. **Double conjunction**: "Although...but" (Thai แม้ว่า...แต่ is normal, English doesn't double up)
2. **Because + so**: "Because I was tired, so I slept" — Thai เพราะ...เลย is common, English uses only one
3. **Run-on sentences**: Connecting too many clauses with "and" without proper punctuation
